#  Exoplanet Transit Detection using Lightkurve

This project demonstrates the detection of exoplanets using NASA Kepler mission data.

##  Project Overview

- Extracted light curves from Kepler Target Pixel Files (TPF)
- Applied detrending to remove noise
- Detected transit signals using Box Least Squares (BLS)
- Folded light curve to reveal periodic transit
- Estimated planetary parameters

##  Tools Used

- Python
- Lightkurve
- NumPy
- Matplotlib
- Google Colab (Cloud Computing)

##  Results

- Clear transit detection
- Estimated orbital period
- Planet radius ratio calculated from transit depth

##  Project Structure

- notebooks/ → analysis notebook
- data/ → processed data
- figures/ → plots and results

##  Key Concepts

- Transit photometry
- Time-series analysis
- Signal detection (BLS)
- Exoplanet parameter estimation

## 📌 Conclusion

This project replicates real astronomical workflows used in exoplanet discovery using open-source tools and cloud computing.
